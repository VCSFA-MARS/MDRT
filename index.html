Welcome to MARS Data Review Tool Project.

This project was started in 2013 to process and review MARS ECS data. The current version parses FCS .delim files and allows plotting up to three time-synchronized subplots at a time. Additional helper functions for quantifying trends, checking boundary violations and many other tasks are included.

Usage and Requirements

The MARS Data Review Tool is designed to be used in MATLAB and does not require special MATLAB toolboxes. Future development efforts will focus on creating a deployable application for individual engineers, leveraging the MATLAB deployment kit.

Minimum Requirements:

  • Matlab R2014b (or greater)
  • Mac OSX 10.6+ or Linux (to run Matlab)
  • MS Windows 7 + (known issues with .delim parsing)
